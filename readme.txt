🏁 F1 Live - Página Web de Fórmula 1

F1 Live es una página web informativa y visual sobre la Fórmula 1. Aquí encontrarás noticias actualizadas, escuderías, pilotos, calendario, clasificación y mucho más.

🔍 Características principales

📰 Noticias destacadas sobre la F1 (temporadas, fichajes, equipos, etc.).

🏎️ Sección de Escuderías 2025 con logos e información visual.

👨‍✈️ Pilotos destacados con imágenes y nombres ordenados por equipos.

📆 Calendario completo de la temporada 2025.

🏆 Tabla de Clasificación de pilotos actualizada.

📸 Galería de imágenes de autos y momentos clave.

📬 Sección de Contacto para que los fans puedan enviar mensajes.

💻 Tecnologías utilizadas

HTML5

CSS3

Google Fonts (Roboto)

Imágenes de Unsplash y otras fuentes públicas

Diseño responsive (adaptado a pantallas móviles)

📁 Estructura del proyecto
F1-Live/
│
├── index.html           → Página principal
├── style.css            → Estilos y diseño
├── script.js            → (opcional) Funcionalidad JS si se desea agregar

Clona o descarga este repositorio.

Abre el archivo index.html en tu navegador.
